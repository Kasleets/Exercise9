Bish Bosh 2.0

Application purpose is to calculate which numbers are divisible by two different values. If a number requires one or the other specified value, then it displays the text equivalent to it. For example 3 and 6 are both divisible by 3 so both will display Bish.

If the provided number fills both required values, then it displays the output as two words together. For example 12 is both divisible by 3 and 4 so it will display BishBosh.

Provide your custom input for Bish, Bosh, and the range to generate the sequence or click on the button to test default values.
